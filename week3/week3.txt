자바의 자료형 : 원시형과 갹체형

객체형 
 - 키로 구분된 데이터 집합이나 복잡한 개체 저장
 - 키 : 문자형, 값 : 모든 자료형 허용
 (빈 객체 생성 방법)
  - 객체 생성자 -> let user = new Object();
  - 객체 리터럴 -> let user = {};

let user = {
    name : "John",
    age : 30
};
-> 객체 user안에 두개의 프로퍼티가 있음
  - alert (user.name);  //프로퍼티 값 읽기
  - user.isAdmin = true;  //프로퍼티 추가
  - delete user.age;  // 프로퍼티 삭제
